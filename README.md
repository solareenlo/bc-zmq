# bc-zmq
ビットコインのフルノードに対してzmqを使って, 新しいトランザクションやブロックの情報を定期購読するシンプルなプログラム.

### Usage
```bash
git clone
cd bc-zmq
// .env_ezampleを参考に, .envを作って, その中にBitcoinのアドレスを代入する.
yarn or npm install
yarn start or npm run start
```
